# Back
A Tshock plugin that teleports you back to your last death position

## How to Install
1. Download the `.dll` file.
2. Put the `.dll` file inside of `/ServerPlugins/`
3. Stop and rerun the server.

## Versions
Back v1.0.1 (Latest)

## Instructions
### Commands
`/back`

### Permissions
`back.back`
